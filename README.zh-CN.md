Mind Your Bruises（注意你的淤青）是一个 Fabric 客户端模组，会根据伤害来源重新着色 Minecraft 原版实体受击闪烁效果。

## 功能

- 在不改变伤害机制的前提下重新着色原版实体受击覆盖效果。
- 将常见伤害类型归入易读的颜色主题。
- 对未匹配或普通物理伤害保留原版红色兜底效果。

兼容性：

- Burn By Soul Fire：安装该模组时，灵魂火燃烧视觉会匹配为蓝色的冰霜主题受击闪烁。

## 运行要求

- Fabric Loader
- Java 21 或更新版本
- 与下载文件匹配的 Minecraft 版本

可选：

- Mod Menu
- YetAnotherConfigLib

## 配置

生成的配置文件位于 Minecraft 实例的配置目录：

```text
<instance folder>/config/mind-your-bruises.json
```
